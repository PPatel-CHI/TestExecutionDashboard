Automation Execution using Electron+Docker+Zalenium

Setup

Steps:

1. Install Docker for windows from docker hub2. pull Zalenium image 3. Run Zalenium: > docker run --rm -ti --name zalenium -p 4444:4444 -v /var/run/docker.sock:/var/run/docker.sock -v /tmp/videos:/home/seluser/videos --privileged dosel/zalenium start 4. Verify if the Zalenium is running in your machine with command: > Docker ps -a5. Navigate to localhost:4444\grid/live6. Update your driver with remoteDriver and trigger run
